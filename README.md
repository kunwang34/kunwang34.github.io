# Kun Wang academic homepage

This repository contains Kun Wang's academic homepage built with the Academic Pages Jekyll template.

## Local structure

- `_pages/about.md`: homepage content
- `_publications/`: publication entries
- `_pages/cv.md`: CV page
- `files/Kun_Wang_CV.pdf`: downloadable CV

## GitHub Pages

Use a repository named `kunwang34.github.io` for the clean URL:

```powershell
git init
git add .
git commit -m "Build Academic Pages homepage"
git branch -M main
git remote add origin https://github.com/kunwang34/kunwang34.github.io.git
git push -u origin main
```

Then enable GitHub Pages from the repository settings if it is not already active.
